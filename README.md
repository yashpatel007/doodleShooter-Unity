# doodleShooter-Unity

Intro:
------
This is a simple ball shooter game made in unity, the game have a sling to which a black ball is attached, this sling
is used to shoot the ball and this black ball is supposed to hit the blue ball in such a way that the blue ball reached the 
specific target.

Prerequisites:
--------------
* You need the UNITY Game Engine which you can download from here : https://store.unity.com
make sure to select the personal edition to download and use it for free.

* you also need a text editor such as sublime text or notepad++ according to your prefrence

Installing:
-----------
In order to download the project open the terminal at folder where you want to copy the repository, then type following command at the terminal.

`git clone https://github.com/yashpatel007/doodleShooter-Unity.git`

It shouls copy the project to target folder.
